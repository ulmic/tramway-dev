## Installation
Add this line to your application's Gemfile:

```ruby
gem 'tramway-event'
gem 'carrierwave'
```

And then execute:
```bash
$ bundle
```
*Gemfile*
```ruby
gem 'tramway-event'
```

Then run:
```shell
rails g tramway:event:install
rails db:migrate
```
