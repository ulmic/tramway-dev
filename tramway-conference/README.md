## Usage
How to use my plugin.

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'tramway-conference'
```

And then execute:
```bash
$ bundle
$ rails g tramway:conference:install
$ rails db:create db:migrate
$ rails tramway:conference:prepare
```
